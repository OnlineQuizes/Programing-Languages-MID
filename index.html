32. What is portability in a language?

Correct Answer: a) Works on many systems
Explanation: Portability means a program can run on different systems with little or no modification. Example: A program written in C runs on Windows, Linux, and macOS with little or no change.

33. What does generality in a language mean?

Correct Answer: a) Supports many kinds of applications
Explanation: Generality means a language can be used for various types of applications. Example: Python can do many tasks ‚Äî web, data, AI, and more.

34. The well-definedness criterion ensures:

Correct Answer: c) Clear and predictable behavior
Explanation: Well-definedness ensures clear and predictable behavior of programs. Example: In Java, dividing by zero always gives an error, not random results.

35. Portability is most strongly influenced by the degree of standardization of the language.

Correct Answer: a) True

36. How does simplicity affect readability?

Correct Answer: a) Greater simplicity increases readability
Explanation: Greater simplicity increases readability. Example: total = price * quantity ‚Üí easy to understand vs t = p * q * (1 + 0) ‚Üí confusing

37. An example of expressivity in a language is:

Correct Answer: c) Doing complex work with simple code
Explanation: Expressivity means doing complex work with simple code. Example: squares = [x*x for x in nums] does a complex task (create a list of squares) in one simple line

38. When two names refer to the same memory location, it's called:

Correct Answer: b) Aliasing
Explanation: Aliasing occurs when two names refer to the same memory location. Example: int x = 10; int *p = &x; // p and x refer to the same memory
